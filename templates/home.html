Weather Data API

Welcome to the Weather Data API documentation. Below you can find the URL formats to access weather data:

1. URL for the station on a specific date:

http://127.0.0.1:5000/api/v1/10/2003-01-23

2. URL for the station over the entire period:

http://127.0.0.1:5000/api/v1/10/

3. URL for the station's annual data:

http://127.0.0.1:5000/api/v1/annual/10/2003
